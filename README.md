# UserStory-M1-W1

An easy made program on python to register a sale, and show the product's name, price and quantiity

## ► How does it work?

° Asks the user to enter the product's name, price and the quantity that's gonna be bought                                                              
° Validates all the data is right before continuing                                                                                                   
° Calculates the total of the sale                                                                                                                  
° Shows the client a message showing all the information of the purchase
